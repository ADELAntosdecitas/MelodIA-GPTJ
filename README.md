# MelodIA Tokio Premium - GPT-J 1.3

Este proyecto despliega una app Flask conectada con GPT-J (EleutherAI/gpt-j-6B).

## Cómo correrlo en local

```bash
pip install -r requirements.txt
python3 app.py
```

## Endpoints

- `/` - Frontend HTML
- `/api/chat` - POST `{ prompt: "..." }`
